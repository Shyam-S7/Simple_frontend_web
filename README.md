# 🥘 Briyani Payaluga - Authentic Restaurant Website

Briyani Payaluga is a sophisticated restaurant frontend application designed to showcase authentic South Indian cuisine. This project focuses on a clean user experience, mobile-first responsiveness, and elegant food presentation.

## 🌟 Overview
This website serves as a digital storefront for **Briyani Payaluga**, featuring a full-service menu, branch locations, and an integrated contact system. It is built with performance and SEO best practices in mind using modern web standards.

## ✨ Key Features
- **Mobile-First Responsive Layout**: Optimized for seamless viewing across smartphones, tablets, and high-resolution desktops.
- **Dynamic Menu Showcase**: A comprehensive, categorized display of breakfast, signature briyanis, and special meal offerings.
- **Intuitive Navigation**: A clean, accessible menu structure that makes it easy for customers to find branches and contact info.
- **Visual Excellence**: High-quality imagery and a curated color palette reflecting a premium dining atmosphere.
- **Action-Oriented Design**: Prominent call-to-action buttons for table bookings and instant inquiries.

## 🛠️ Technology Stack
- **Structure**: Semantic HTML5 markup for enhanced search engine optimization and screen reader accessibility.
- **Styling**: Robust Vanilla CSS3 using a modern design system with reusable CSS variables and flexbox/grid layouts.
- **Typography**: A combination of **Inter** and **Outfit** Google Fonts for a readable and professional look.
- **Iconography**: Scalable vector icons from **Font Awesome 6.5.0** for a modern, crisp interface.

## 📂 Project Structure
- `index.html`: The central hub and landing page of the application.
- `menu.html`: A detailed catalogue of dishes with pricing and descriptions.
- `branches.html`: Information regarding physical restaurant locations.
- `about.html`: Insights into the brand's culinary journey and values.
- `contact.html`: Contact details, business hours, and reservation links.
- `assets/css/style.css`: The core design engine containing all styles.
